# Viva Notes

## One-line project summary

This project detects whether an uploaded face image is real or deepfake using a public pretrained model, with confidence scores, history, reports, and visual explanations.

## Key talking points

- It uses transfer learning and a pretrained checkpoint instead of training from scratch.
- It is designed for Windows laptops without GPU.
- It includes a full Flask web application with proper UI and error handling.
- It saves prediction history and generates downloadable reports.
- It provides explainability with a Grad-CAM heatmap when available.

## Why it is production-ready

- Clear folder structure
- Input validation
- Persistent history storage
- Reusable backend utilities
- Responsive frontend
- Downloadable results

## Limitation to mention honestly

Deepfake detection is a screening task, not a forensic verdict. Accuracy can drop when the uploaded image is heavily compressed, cropped, blurred, or generated by newer AI models not present in the training distribution.
