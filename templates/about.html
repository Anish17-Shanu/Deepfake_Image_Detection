Why This Model

Selection rationale

A public XceptionNet checkpoint with robust image-only performance was not as straightforward to obtain as a reusable, documented image classifier. The selected ViT checkpoint has a published self-reported accuracy of 99.27%, a standard Hugging Face repository, and clear downloadable files. That makes it stronger for a production-style student project than an undocumented weight file.

Preprocessing

How inference works

  • Images are converted to RGB and resized to 224 × 224.
  • Pixel values are rescaled and normalized using the model processor.
  • The model outputs class probabilities for Real and Fake.
  • Grad-CAM overlay is created from the last transformer block when available.
Expected Performance

Accuracy and limitations

The public checkpoint reports 99.27% accuracy on its evaluation set. In practice, real-world accuracy can be lower because of camera noise, compression, crop quality, and concept drift from newer AI-generated images.
